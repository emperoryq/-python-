# -python-
全文代码基于python3.8.1

movieParadise.py为爬取电影天堂网站一类电影的信息

AllmovieParadise.py为爬电影天堂网站全站电影信息

在movieParadise.py中改变depth的数值即可修改爬取的页数

在AllmovieParadise.py中，depth为修改同类电影的爬取页数，style为爬取了多少种不同类电影
